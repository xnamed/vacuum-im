1. Druk op «Open registratie website» om registratie pagina te openen in je web browser.
  2. Voer de gewenste gebruikersnaam in «Gebruikersnaam:» veld.
  3. Voer gewenste paswoord in «Paswoord:» veld.<br>
  4. Solve the problem after «What is» below and type the answer into field next to it.
  5. Press «Register!» button to proceed registration.
  6. If registatration is successful, close your web browser, or try different user name otherwise.
  7. Druk «Volgende» om verder te gaan naar de referenties pagina.
